# ZunCafe

Coffee shop e-commerce. Next.js, Material-UI, JSON for products and orders.

- Home, Products, Product detail, Cart, Checkout
- Cart in browser. Orders in data/orders.json

Run: `npm install` then `npm run dev`. Open http://localhost:3000

Edit data/products.json for products. Orders go to data/orders.json.
